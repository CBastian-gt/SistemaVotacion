=============================
LEEME.TXT - Instrucciones de Instalación
=============================

Versión de PHP: 8.2.4
Base de datos: MariaDB 10.4.28

1. Requisitos previos:
   - Asegúrate de tener instalado PHP 8.2.4 en tu servidor. Puedes descargar la última versión de PHP desde el sitio web oficial de PHP (https://www.php.net/downloads.php).
   - Configura un servidor web compatible con PHP. Recomendamos utilizar XAMPP, que incluye Apache y MySQL. Puedes descargar XAMPP desde su sitio web oficial (https://www.apachefriends.org).

2. Descarga el proyecto:
   - Clona o descarga el repositorio del proyecto desde https://github.com/CBastian-gt/SistemaVotacion
   - Extrae los archivos en el directorio de tu servidor web local.

3. Configuración de la base de datos:
   - Abre el panel de control de XAMPP y asegúrate de que los servicios de Apache y MySQL estén activos.
   - Accede a phpMyAdmin desde el panel de control de XAMPP o ingresando la siguiente URL en tu navegador: http://localhost/phpmyadmin
   - Crea una nueva base de datos con el nombre que desees para el proyecto.
   - Importa el archivo `formulariovotacion.sql` incluido en el proyecto en la base de datos recién creada.
    Puedes hacerlo desde phpMyAdmin seleccionando la base de datos y utilizando la opción de importar.

4. Iniciar la aplicación:
   - Abre un navegador web y accede a la dirección local de tu servidor web, por ejemplo: http://localhost/proyecto.
   - La aplicación debería cargarse y estar lista para su uso.

¡Enhorabuena! Has completado la instalación y configuración del proyecto.
Ahora puedes disfrutar de todas las características de la aplicación.

Versión de PHP utilizada: 8.2.4
Base de datos utilizada: MySQL
